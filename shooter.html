Shooter

This project is based on TheKiwiCoder's work using Synty assets. With Unity 6 introducing Cinemachine 3, it was no longer possible to follow his tutorial step-by-step to recreate the laser gun setup featuring a triple-layered weapon recoil system. I developed a straightforward upgrade path to Cinemachine 3 that maintains the original three-layer recoil approach.

Weapon Recoil

The weapon recoil system combines three elements: impulse effect recoil, animation recoil, and camera movement recoil. Impulse effect recoil uses the Cinemachine Impulse Source to create the classic camera shake. Animation recoil layers a firing animation on top of the character stance. Camera movement recoil is the most distinctive elementâ€”it uses Cinemachine Orbital Follow to move the camera in a controlled pattern, such as an S-shaped climb. This forces the player to actively counter the recoil by pulling their aim downward to stabilize. 

// Smoothly apply recoil displacement over the step time
private void PlaybackRecoil()
{
    if (recoilTimer > 0)
    {
        CinemachineOrbitalFollow.VerticalAxis.Value -=
        (verticalRecoil * patternScale.y * Time.deltaTime) / recoilStepTime;
        
        CinemachineOrbitalFollow.HorizontalAxis.Value -=
        (horizontalRecoil * patternScale.x * Time.deltaTime) / recoilStepTime;
        
        recoilTimer -= Time.deltaTime;
    }
}
